#尺寸相关

```obj
    首先表明，这里不讨论不同规格的设备的尺寸，只讨论UI控件的尺寸。也就是任意一个UI控件的frame和size属性
```

###1.frame、origin、size的基本使用

```obj

  在iOS开发中，用宽、高(width、height)属性去描述UI控件的大小,在一个屏幕中,用二维坐标系去表示一个控件的位置(x,y),
这两个属性是控制UI控件尺寸的常用属性,本质上它们分别归属于CGSize、CGPoint两个结构体，分别是UI控件的frame属性下的
size属性和origin属性；下面用一段代码演示： 

// 首先创建一个View，并且设置它的位置和大小
 UIView *testView = [UIView new];

// CGPointMake 这个方法用于创建一个坐标，参数分别是x值，y值
CGPoint testViewP = CGPointMake(10, 10);

// CGSizeMake 这个方法用于创建一个坐标，参数分别是width，height
CGSize testViewS = CGSizeMake(50, 50);

// 用UI控件的frame属性来设置UI控件的位置和大小
// CGRectMake这个方法用于创建一个描述UI控件位置和大小的结构体
// 第一个参数、第二个参数均用于描述 控件所在位置，分别描述二维坐标系的x值、y值
// 第三个参数、第四个参数均用于描述 控件大小，分别描述二维坐标系的宽width、高height
testView.frame = CGRectMake(testViewP.x, testViewP.y, testViewS.width, testViewS.height);

然后是获取这个UI控件的位置和大小
//1.获取控件的位置

 CGPoint testViewPozition = testView.frame.origin;

// x值

 CGFloat testViewX = testViewPozition.x;

// 上面这段代码等价于

// testViewX = testView.frame.origin.x;

// y值

 CGFloat testViewY = testViewPozition.y;

// 上面这段代码等价于

// testViewY = testView.frame.origin.y;

// 2.获取控件的大小

 CGSize testViewSize = testView.frame.size;

// width值

 CGFloat testViewW = testViewSize.width;

// 上面这段代码等价于

// testViewW = testView.frame.size.width;

// height值

CGFloat testViewH = testViewSize.height;

// 上面这段代码等价于

// testViewW = testView.frame.size.height;



```

###2.相关知识的详细介绍

```obj
1、数据类型：

CGFloat: 浮点值的基本类型

CGPoint: 表示一个二维坐标系中的点

CGSize: 表示一个矩形的宽度和高度

CGRect: 表示一个矩形的位置和大小

typedef float CGFloat;// 32-bit

typedef double CGFloat;// 64-bit



struct CGPoint {

 CGFloat x;

 CGFloat y;

};

typedef struct CGPoint CGPoint;



struct CGSize {

 CGFloat width;

 CGFloat height;

};

typedef struct CGSize CGSize;



struct CGRect {

 CGPoint origin;

 CGSize size;

};

typedef struct CGRect CGRect;

注意：CGRect数据结构的高度和宽度可以是负数。例如，一个矩形的原点是[0.0，0.0]和大小是[10.0,10.0]。这个矩形完全等同原点是[10.0，10.0]和大小是[-10.0，-10.0]的矩形。

2、使用值来创建几何元素的方法

CGPointMake CGRectMake CGSizeMake

CGPoint CGPointMake (

 CGFloat x,

 CGFloat y

);



CGSize CGSizeMake (

 CGFloat width,

 CGFloat height

);



CGRect CGRectMake (

 CGFloat x,

 CGFloat y,

 CGFloat width,

 CGFloat height

);

 CGFloat ten=10.0f;

 CGPoint point = CGPointMake(0.0f, 0.0f);

 CGSize size = CGSizeMake(10.0f, 10.0f);

 CGRect rect = CGRectMake(point.x, point.y, size.width, size.height);

 NSLog(@"ten: %f", ten);

 NSLog(@"point: %@", NSStringFromCGPoint(point));

 NSLog(@"size: %@", NSStringFromCGSize(size));

 NSLog(@"rect: %@", NSStringFromCGRect(rect));
```
